tcp port - Идентификатор пакета, передаваемого по протоколу TCP/IP. Номер порта используется для маршрутизации данных внутри одного хоста и обозначает процесс, которому необходимо доставить конкретный пакет.

socket -  название программного интерфейса для обеспечения обмена данными между процессами. Процессы при таком обмене могут исполняться как на одной ЭВМ, так и на различных ЭВМ, связанных между собой только сетью. Сокет — абстрактный объект, представляющий конечную точку соединения.

Сетевой сокет-это программная структура в сетевом узле компьютерной сети, которая служит конечной точкой для отправки и приема данных по сети.
